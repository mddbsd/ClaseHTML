FlexBox

El tipo flex, o flexbox, es un modulo de organizacion de elementos HTML. Con flexbox, creamos "cajas" flexibles que se iran acomodando de manera responisva, osea , que cambian su posicion y tamaño de manera dinamica

Un flexbox siempre contiene estos 2 elementos:

  • un Contenedor flexible - el elemento div padre
  • Flex Items - los items dentro del contenedor div
Direccion del flexbox

La propiedad flex-direction especifica la direccion de rendrerizado de los intems DENTRO del flex container
Esta propiedad tiene los siguientes valores

  • row - fila
  • column - columna
  • row-reverse - fila invertida
  • column-reverse - columna invertida
