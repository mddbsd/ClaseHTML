Hypervínculos

Un hypervínculo es la herramienta más importante de la escritura +ón con hypertexto-
Estos nos permiten desplazarnos a travez de un documento o entre documentos. Funcionan como accesos directos de windows

Para crear hipervinculos utilizamos la etiqueta <a>. Esta etiqueta contiene un atributo llamado href donde escribimos la ruta al documento que queremos apuntar o al id del elemento HTML en el mismo documento
